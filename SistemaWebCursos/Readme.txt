Carpeta contenedora de los archivos del proyecto

*bin 	     => Archivos de paquetes de C#
*Controllers => Controladores creador por Entity Framework
*Data 	     => Archivos referentes a la BD
*Models      => Clases de los objetos de la BD
*obj	     => Archivos de depuracion y paquetes nuget
*Views 	     => Paginas web aspx del sistema
*wwwroot     => Hojas de estilos css, Imagenes png, Funciones js, Librerias

+appsetting.json => Archivo contenedor de los datos de la conexion con la BD
+Startup.cs 	 => Archivo que define los metodos de configuracion de inicio
